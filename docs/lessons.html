Week 1, Day 1

Before class:

  • Install git
  • Get GitHub account
  • Sign up for GitHub Education Pack
  • Install GitKraken

During class:

  1. Go over course outline and syllabus
  2. Set aside some time in the beginning to finish and troubleshoot above.
  3. Go over git and GitHub basics: what are they, why useful, etc.
  • take from the beginning of https://www.lucasnell.com/resources/github-rstudio/
  • another good source of info: https://happygitwithr.com/
  1. Fork template repo and clone to local machine: https://github.com/lucasnell/pranga-template
  2. Go over Markdown format: https://www.markdownguide.org/basic-syntax/
  3. Students edit README.md in their new repo, commit, and push to GitHub
  4. If doing solitary projects - 15 minutes to hash out their ideas and talk to us about their ideas.
